✅ AUTHENTICATION (If Prompted)
═══════════════════════════════════════════════════════════════════════════════

GitHub will prompt for authentication:

Option 1 - Browser Authentication (Easiest)
  • Click "Authorize" in browser window
  • Returns to PowerShell automatically
  • Done!

Option 2 - Personal Access Token
  1. Go: https://github.com/settings/tokens
  2. Click: Generate new token
  3. Select: repo, read:user
  4. Generate and copy
  5. Paste when GitHub asks for password
  6. Done!

Option 3 - GitHub CLI (Most Convenient)
  1. Download: https://cli.github.com/
  2. Install
  3. Run: gh auth login
  4. Authenticate
  5. Retry: git push -u origin main
