📞 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

Problem: Blank page at localhost:3000
Solution: 
   • Wait 10 seconds for compilation
   • Refresh browser (F5)
   • Check terminal for errors

Problem: "Cannot POST /api/analyze-resume"
Solution:
   • Verify backend running on port 5000
   • Check terminal 1 for "Server is running" message

Problem: CSS not loading
Solution:
   • Hard refresh browser (Ctrl+Shift+R)
   • Clear browser cache
   • Restart frontend server

Solution:
