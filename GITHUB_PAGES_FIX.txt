Error: "The site configured at this address does not contain the requested file"

Cause: GitHub Pages is enabled but not properly configured for your repository

Your Repository: https://github.com/vignesh-muniyandi/AI-resume-checker

✅ SOLUTION OPTIONS
══════════════════════════════════════════════════════════════════════════════════

OPTION 1: Disable GitHub Pages (Recommended if you only want code repo)
───────────────────────────────────────────────────────────────────────────────

If you just want your code on GitHub (not a live website):

1. Go to: https://github.com/vignesh-muniyandi/AI-resume-checker/settings
2. Navigate to: Pages (left sidebar)
3. Under "Build and deployment":
   - Source: Select "None" or "Disabled"
4. Save
5. Done! The GitHub Pages error will disappear


OPTION 2: Enable GitHub Pages Properly (If you want live website)
───────────────────────────────────────────────────────────────────────────────

If you want to host your React app on GitHub Pages:

Step 1: Update package.json (Frontend)
──────────────────────────────────────────────────────────────────────────────

In frontend/package.json, add:

"homepage": "https://vignesh-muniyandi.github.io/AI-resume-checker",


Step 2: Build the Frontend
──────────────────────────────────────────────────────────────────────────────

Run in PowerShell:

  cd d:\Identify\frontend
  npm install
  npm run build

This creates a /build folder with static files


Step 3: Use gh-pages Package (Easy Method)
──────────────────────────────────────────────────────────────────────────────

Install gh-pages in frontend:

  npm install --save-dev gh-pages

Add scripts to frontend/package.json:

  "predeploy": "npm run build",
  "deploy": "gh-pages -d build"

Run:

  npm run deploy

This automatically:
  • Builds your React app
  • Creates gh-pages branch
  • Pushes to GitHub Pages
  • Your site goes live!


Step 4: Configure GitHub Pages
──────────────────────────────────────────────────────────────────────────────

1. Go to: https://github.com/vignesh-muniyandi/AI-resume-checker/settings
2. Navigate to: Pages
3. Under Source: Select "Deploy from a branch"
4. Select branch: gh-pages
5. Select folder: root
6. Save


Your site will be live at:
  https://vignesh-muniyandi.github.io/AI-resume-checker/
